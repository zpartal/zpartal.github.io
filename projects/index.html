My Projects

Below is a sample of some of the projects I have worked on. These projects range from apps for Windows 8 and Windows Phone to interesting programming challenges from Project Euler to some homework projects I have worked on during school.

Windows 8 Apps

Piano Fun!

Piano fun is virtual piano app for Windows 8 that I wrote during a 24 hour Microsoft sponsored hackathon at Texas A&M in February 2013. The app even lets you select through multiple octaves for the ability to play a large range of notes. The app can be downloaded from the Windows 8 Store.

Hangman Fun!

Hangman Fun is an app I wrote for Microsofts App Madness program in March 2013. It is the classic hangman game designed for Windows 8 touch screen devices. The app can be downloaded from the Windows 8 Store.

  • Download

Tic Tac Attack!

Tic Tac Attack is an app I wrote for Microsofts App Madness program in March 2013. It is the classic tic tac toe game designed for Windows 8 touch screen devices. The app features three modes of game play. You can play against another human, against an easy computer that chooses random moves and against a hard computer which tries its best to win. The app can be downloaded from the Windows 8 Store.
